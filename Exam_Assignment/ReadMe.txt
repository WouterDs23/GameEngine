Ik heb gekozen om te werken met smart pointers dus zorgerde ervoo dat ik geen destructors moest gebruiken en dat maakte op bepaalde momenten het leven toch gemakkelijker.

Ook heb ik gebruik gemaakt van het command pattern, State pattern , Component pattern , Game Loop en Update Method

Ook heb een een map loader die een tekst document kan inladen

Mijn project bevat 2 levels en 3 verschillende game modes

Ik heb ook een standaard wander en seek behaviour
